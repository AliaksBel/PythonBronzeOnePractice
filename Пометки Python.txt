Команды Phython
функция:
print - вывести результат на экран - в круглых скобках в функцию передаются аргументы - данные, которые функция использует для работы.
символы апострофа (', одинарной кавычки) означают, что данный аргумент - текстовый, т. е. относится к строковому типу данных.
особенность функции print - она умеет выводить любые данные, преобразуя их автоматически в текст.

символ \ играет роль экранирующего символа - управляющей последовательностью
\n - перевод строки;
\t - символ табуляции;
\\ - символ обратного слеша;
\' - символ апострофа (применим в том числе строках, ограниченых ');
\" - символ апострофа (применим в том числе строках, ограниченых ").

С помощью именованного аргумента sep мы смогли настроить разделитель между выводимыми функцией print элементами.
Прежде чем двигаться дальше, давайте подробнее поговорим об именованных аргументах. При их использовании нужно иметь ввиду следующее:

- именованных аргументов, как и обычных, может быть несколько;
- как и обычные, именованные аргументы определаются при создании функции;
- передавать в функцию можно только определенные в ней именованные аргументы;
- именованные аргументы должны указываться в конце списка аргументов;
- именованные аргументы могут иметь значение по умолчанию, применяемое в ситуации, когда аргумент не передается.
- Для функции print определено несколько именованных аргументов.
